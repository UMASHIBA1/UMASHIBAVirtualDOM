# UMASHIBAVirtualDOM
This is virtual dom for studying
## Usage 使い方

```
git clone https://github.com/UMASHIBA1/UMASHIBAVirtualDOM.git
cd .\UMASHIBAVirtualDOM
yarn install
yarn start
```
